    Open Source Arms Race

    The following text is a translation of the original English article, performed by ChatGPT (gpt-3.5-turbo) and this Jekyll plugin:

    一些公司大量投资于开源软件项目，而其他一些公司仍然持怀疑态度，并远离这一趋势。像谷歌、IBM和微软这样的慈善家们究竟从中得到了什么好处呢？为什么要花钱投资于不属于他们自己，而是共享于所有开发者的东西？难道他们不明白他们编写的代码可能会被竞争对手使用吗？看起来他们确实明白，但却无法做任何事情去改变这一现状。

    正如David Mytton所指出的，个人出于利他主义的原因发布开源项目，而公司背后则隐藏着真正的战略原因，这在开源的温暖和模糊光芒之后不为人所知。我们指的不仅仅是使用开源产品，这是现今每个企业所做的，而是构建这些产品，让每个人都可以免费使用。

    首先，让我们来看看公司在开源方面的投资方式，包括其他方法。

    贡献者们。例如，谷歌鼓励其员工开放他们的代码，甚至为此设有一套审批程序。似乎其他公司也有类似的做法。最近发表的一些研究表明，来自微软（4.3k 可见的 GitHub 贡献者）、IBM（2.3k）、谷歌（1.2k）和其他公司的成千上万名员工积极地为他们的 GitHub 存储库做出贡献。换句话说，这些受雇于谷歌的人们编写的代码并不属于谷歌。

    活动。除了Google Cloud Next、AWS re:Invent和Microsoft Ignite这些纯粹的促销活动外，科技巨头还赞助社区驱动的会议，如DockerCon、KubeCon、FOSDEM等等。

    基金会。Apache、Linux、Eclipse、GNOME、OSI等项目得到Google、IBM、华为、微软、Oracle等公司的赞助。

    收购。他们收购制造开源产品的初创企业，例如RedHat（以320亿美元被IBM收购）、MongoDB（40亿美元）、Docker（10亿美元）、Elastic（60亿美元）、Ansible（1亿美元）、MySQL（10亿美元）、Nginx（6.7亿美元）等等。顺便说一下，市场正在蓬勃发展。

    捐赠。例如，谷歌向Fastlane和Kubernetes捐赠；IBM向Node.js和TensorFlow捐赠；亚马逊赞助Rust等等。

    接下来，问题是这如何帮助谷歌和其他公司增加利润。例如，谷歌云平台如何计划利用开源作为武器击败亚马逊AWS？以下是其中一些原因：

    销售。他们赞助的产品聚集了大量的用户社群，这些用户知道如何使用这些产品。其中一些用户是决策者。例如，Google的容器引擎是Kubernetes的托管版本，Kubernetes是一个开源产品。当Kubernetes的用户需要托管版本时，很可能会成为Google的客户。再举一个例子，TensorFlow也可以在Google云平台上运行。

    影响力。在现代软件开发世界中，开源是关键媒体，程序员通过它学习新的想法、设计原则和架构解决方案。他们倾听开源项目，因为他们对其的信任远远超过对谷歌、IBM或微软的信任。科技公司很聪明，但他们的决策是基于金钱原因，而开源项目则更加自由地做正确的事情，而不是盈利。当开源项目被大型企业收购时，如Oracle收购Java或微软收购GitHub（虽然不是真正的开源故事，但非常接近），程序员通常会变得非常怀疑。因此，对于公司来说，赞助开源并在幕后保持低调，而不是通过企业媒体如MSDN或OPN直接与开发者交流，是更加明智的选择。

    品牌建立。成为开源的一部分是对公司公众形象的直接贡献。谷歌不再是一个声誉可疑的广告销售垄断者，而是最大的软件慈善家之一。微软通过大规模开源代码和收购GitHub正在迎头赶上。

    保留。对于最优秀的软件工程师来说，成为一个可见的开源软件产品的积极参与者是一种极好的内在动机。当你的公司参与开源业务时，你会更加忠诚并对公司的长期留任感兴趣。我无法证明这个观点，但这是我的个人观察。

    培训。如今，最好的技术往往是由为开源项目做出贡献的人们免费创造的（他们的薪水来自其他地方），这已经不是什么秘密。众所周知，为了成为一名更好的工程师，必须与最优秀的人一起工作。开源项目恰恰是结识更有经验、经常更有才华的程序员并向他们学习的最佳场所。公司通过派遣员工参与开源项目，提高他们的技能，而无需进行任何内部培训计划。

    因此，看起来开源是科技市场和人才战争中非常强大的工具。聪明的公司发展他们的开源实力，赢得声誉并获得我们的信任。我预测这种趋势不仅会持续下去，还会升级。我们程序员是他们的推动力；他们需要我们作为用户、推广者、影响者和产品创作者。

    他们长期以来对开源领域不关注，现在他们正在为此而战，试图获得最大的份额。

    我们软件开发者是这片领地的主宰！

    因此，如果你是一名程序员，现在就开始自己的项目吧。迟早谷歌会敲开你的门，给你一大笔支票，只为了比微软更早实现这一目标。
