    Open Source Arms Race

    The following text is a partial translation of the original English article, performed by ChatGPT (gpt-3.5-turbo) and this Jekyll plugin:

    有些公司大量投资于开源软件项目，而其他公司仍然持怀疑态度，并远离这一趋势。像谷歌、IBM或微软这样的慈善家们会得到什么好处呢？为什么要花钱在一些不属于他们自己、而是与我们所有开发者共享的东西上？难道他们不明白他们编写的代码可能会被竞争对手使用吗？看起来他们确实明白，但却无能为力。

    如David Mytton所指出，个人出于利他主义而发布开源项目，而公司则在开源的温暖和亲切之光背后隐藏着真正的战略原因。我们所指的并不只是使用开源产品，这是现今每个企业都在做的事情，而是建立这些产品以便每个人都可以免费使用。

    首先，让我们来看看公司在开源项目中的投资方式，其中包括 其他方法。

    贡献者。例如，谷歌鼓励其员工开放源代码，甚至为此设有一个批准程序。似乎其他公司也有类似的做法。最近一项研究表明，数千名（！）来自微软（4.3k 可见 GitHub 贡献者）、IBM（2.3k）、谷歌（1.2k）和其他公司的员工积极贡献于他们的 GitHub 仓库。换句话说，这些人在谷歌的支付下编写的代码并不属于谷歌。

    活动。除了Google Cloud Next、AWS re:Invent和Microsoft Ignite这些纯推广活动之外，科技巨头还赞助由社区主导的会议，如DockerCon、KubeCon、FOSDEM等等。

    基金会。Apache、Linux、Eclipse、GNOME、OSI以及其他由Google、IBM、华为、微软、甲骨文等赞助。

    收购。他们收购制造开源产品的初创公司，例如RedHat（被IBM以320亿美元收购），MongoDB（40亿美元），Docker（10亿美元），Elastic（60亿美元），Ansible（1亿美元），MySQL（10亿美元），Nginx（6.7亿美元），等等。顺便说一下，这个市场正在蓬勃发展。

    捐赠。例如，Google向Fastlane和Kubernetes提供捐赠；IBM向Node.js和TensorFlow提供捐赠；Amazon赞助Rust；以及等等。

    接下来的问题是，这如何帮助谷歌和其他公司提高业务的盈利能力。例如，谷歌云平台计划如何利用开源作为武器来击败亚马逊AWS？以下是其中的一些原因（链接：https://simplabs.com/blog/2019/11/11/why-companies-invest-in-oss/）：

    销售。他们赞助的产品吸引了大量的用户社群，这些用户知道如何使用这些产品。其中一些用户是决策人员。例如，谷歌的容器引擎是Kubernetes的托管版本，它是一个开源产品。当Kubernetes用户需要托管版本时，他们很可能成为谷歌的客户。另一个例子是TensorFlow，它也可以在谷歌云平台上运行。

    影响力。在现代软件开发世界中，开源是关键媒介，程序员通过它学习新思想、设计原则和架构解决方案。他们倾听开源项目，因为他们更加信任它们，而不是Google、IBM或Microsoft。科技公司很聪明，但他们的决策是基于经济原因，而开源项目则更加自由地追求正确的事情，而不是追求利润。当开源项目被大型企业收购时，如Java被Oracle收购或GitHub被Microsoft收购（虽然不是真正的开源故事，但非常接近），程序员通常会持怀疑态度。因此，对于公司来说，赞助开源并在幕后保持低调比直接通过企业媒体如MSDN或OPN与开发人员进行对话更加明智。

    品牌推广。成为开源的一部分直接对公司的公众形象做出贡献。Google不再是一个有争议声誉的广告销售垄断者，而是最大的软件慈善家之一。微软通过大规模的代码开放和对GitHub的收购正在迎头赶上。

    保持。对于最优秀的软件工程师来说，成为一个知名开源软件项目的积极参与者是一种极好的内在动机。当你的公司参与开源业务时，你会更忠诚并且对公司的长期留任更感兴趣。我无法证明这个观点，但这是我的个人观察。

    培训。众所周知，如今最好的技术是由为开源项目做出贡献的人们免费创造的（他们在其他地方获得薪水）。众所周知，要成为更好的工程师，就必须与最优秀的人一起工作。开源项目通常是遇到经验丰富、往往更有才华的程序员，并从他们那里学习的最佳场所。公司通过派遣员工参与其中，提升他们的技能，而无需进行任何内部培训计划。

    因此，看起来开源是科技市场和人才战争中非常强大的工具。聪明的公司会培养他们的开源实力，赢得声誉并获得我们的信任。我预测这个趋势不仅会继续下去，而且会升级。我们程序员是他们的推动力；他们需要我们作为用户、推广者、影响者和产品制造者。

    他们长期以来都没有关注开源领域，现在却在为此而奋斗，试图收购最大的份额。

    我们软件开发者是这个领域的精英！

    因此，如果你是一名程序员，现在就开始自己的项目吧。迟早Google会找上门来，给你开出一张大额支票，只是为了比微软早一步。
