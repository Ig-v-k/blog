    Simplified GitHub Login for a Ruby Web App

    The following text is a partial translation of the original English article, performed by ChatGPT (gpt-3.5-turbo) and this Jekyll plugin:

    你知道什么是OAuth登录吗？当你的用户点击“登录”后，会被重定向到Facebook、Twitter、Google或其他网站，这些网站会识别他们的身份。然后他们会返回到你的网站，你就知道他们是谁了。这对他们来说非常方便。对你来说也很方便，因为你不需要实现登录功能，也不需要将他们的凭据保存在数据库中。我创建了一个简单的Ruby gem，只针对GitHub简化了这个操作。下面是它的工作原理。

    首先，您需要在GitHub上注册您的应用程序，如此页面所述。这是使用Sinatra的工作方式，但您可以在任何框架中进行类似操作。

    首先，在全球范围内的某个地方，在应用程序开始之前：

    require 'glogin'
configure do
  set :glogin, GLogin::Auth.new(
    # You get this from GitHub, when you register your
    # web application:
    client_id,
    # Make sure this value is coming from a secure
    # place and is NOT visible in the source code:
    client_secret,
    # This is what you will register in GitHub as an
    # authorization callback URL:
    'http://www.example.com/github-callback'
  )
end

    接下来，对于所有的网页，我们需要解析一个cookie，如果存在的话，并将其转换为一个用户。

    require 'sinatra/cookies'
before '/*' do
  if cookies[:glogin]
    begin
      @user = GLogin::Cookie::Closed.new(
        cookies[:glogin],
        # This must be some long text to be used to
        # encrypt the value in the cookie:
        encryption_secret
      ).to_user
    rescue OpenSSL::Cipher::CipherError => _
      # Nothing happens here, the user is not logged in.
      cookies.delete(:glogin)
    end
  end
end

    接下来，我们需要一个GitHub OAuth回调的URL：

    get '/github-callback' do
  cookies[:glogin] = GLogin::Cookie::Open.new(
    settings.glogin.user(params[:code]),
    # The same encryption secret that we were using above:
    encryption_secret
  ).to_s
  redirect to('/')
end

    最后，我们需要一个登出的URL:

    get '/logout' do
  cookies.delete(:glogin)
  redirect to('/')
end

    还有一件事就是你需要用于前台页面的登录URL。在这里：

    settings.glogin.login_uri

    对于单元测试，您可以将空字符串提供给secret参数的GLogin::Cookie::Open和GLogin::Cookie::Closed方法，这样加密功能将被禁用：来自cookie的任何内容都将被信任。为了方便测试，可以在查询字符串中提供用户名，例如：

    http://localhost:9292/?glogin=tester

    为了实现这一点，建议您添加这行代码（查看zold-io/wts.zold.io中的示例）。

    require 'sinatra/cookies'
before '/*' do
  cookies[:glogin] = params[:glogin] if params[:glogin]
  if cookies[:glogin]
    # same as above
  end
end

    我在Sinatra上（都是开源的）的sixnines、0pdd和Zold网络应用中使用这个gem。
