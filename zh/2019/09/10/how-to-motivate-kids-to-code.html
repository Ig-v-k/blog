    How to Motivate Kids to Code

    The following text is a partial translation of the original English article, performed by ChatGPT (gpt-3.5-turbo) and this Jekyll plugin:

    几天前，我收到了一封电子邮件。邮件中说：“我不是程序员，我是两个孩子（9岁和14岁）的妈妈。他们似乎对计算机很感兴趣，但主要是玩游戏。你有什么建议可以帮助他们在科技行业找到职业？”虽然我不是育儿专家，但我经常收到类似的请求。很高兴看到有人意识到玩《侠盗猎车手》和Java编程之间的区别。但很遗憾他们不知道如何激励孩子。我也不知道，但我可以试着猜测一下。

    我无法代表每个人，但对于我来说，仅仅为了“学习新事物”而学习根本不有趣。我并不享受变得更聪明的过程。我喜欢创造东西。如果我需要学习新的知识来实现新事物，我会去学习并且会喜欢学习，因为它会带来实际的结果：新的事物。

    例如，Kotlin 已经在市场上存在了几年了。它是一种新的编程语言。我确实意识到它可能最终会取代Java，所以我需要学习它。但是我无法想象自己会去阅读文档并花几天时间学习它。这太无聊了。因为它没有任何意义。嗯，它会让我知道如何使用Kotlin编码，但是如果我已经可以使用Java编码了，那有什么意义呢？为了学习Kotlin，我需要一个新的项目，然后用Kotlin编写它。我将在使用它的过程中学习这个语言。

    我的观点是，为了激励自己学习Kotlin，你需要给我一个目标，在我学会Kotlin之后，利用这些新知识实现这个目标。也许，而且很可能，你的孩子也需要类似的东西：一个有形的激励因素。他们将会在达到目标后看到的东西，这将让他们感到开心，因为他们已经学了那些枯燥的Java和Kotlin语言。对于一个九岁的孩子来说，这样的奖励可能是什么样的呢？

    那么，这些中的其中一个怎么样（也许并不是所有人都适用，但你可以理解这个想法）：

    你能制作一个iPhone手机游戏并在苹果商店上发布吗？

    你可以为我创建一个博客吗？

    你能做一个在线商店来卖我的手工饼干吗？

    你能开发一个应用来帮助我追踪健身锻炼的结果吗？

    你能在下一个黑客马拉松中赢得1000美元吗？

    你能获得Oracle Java证书吗？

    你能在学校的网站中找到安全漏洞吗？

    你能制造一个可以清洁地板的机器人吗？

    其实，这就是我学到所有关于计算机的知识的方式。我大约10岁的时候开始学习编程。我总是有任务在面前，对我自己和周围的人来说，这总是一些有趣的事情。换句话说，我一直对目标保持着关注。我大部分的项目现在已经不再存在了，特别是我小时候和学生时期创建的项目（对此我感到非常难过）。然而，结果是显而易见的：我现在在科技行业，我可以用Java编程。

    所以不要期望你的孩子喜欢学习。期望他们对构建事物和获得金钱利润感兴趣。作为他们的父母，你的工作是帮助他们确定正确的目标—这是孩子们通常失败的地方，因为他们没有足够的经验来判断什么是更可实现的，哪些是最大的风险。我记得当时我尝试创建一个能够像人类一样说话的机器人。那时我13岁。显然，我那时需要帮助：有人告诉我这个目标实际上是不可实现的，而做一些更简单和更实用的事情会更好地应用我的努力。

    不要通过告诉孩子们该学什么、每天要学多少小时、去哪所学校以及读哪些书来过度干涉他们。相反，要像优秀的经理一样，设定目标并确保每个人都清楚奖惩措施是什么。

    类似的情况也是你的孩子需要的。衡量他们的进步，并帮助他们找到足够复杂和有趣的目标。除此之外，确保他们理解目标实现后的好处，而且…呃，你必须找到一种方式来惩罚他们的失败。不要因为他们愚蠢或懒惰而惩罚他们。唯一值得惩罚的是违背达成结果的承诺。

    当然，这是他们对自己做出的承诺。
