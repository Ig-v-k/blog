    How to Motivate Kids to Code

    The following text is a partial translation of the original English article, performed by ChatGPT (gpt-3.5-turbo) and this Jekyll plugin:

    几天前我收到了一封电子邮件。“我不是程序员。我是两个孩子的妈妈，他们分别是9岁和14岁。他们似乎对计算机很感兴趣，但他们主要玩游戏。你会推荐我做些什么来帮助他们在科技行业中有个职业吗？”我不是专家，但我经常收到类似的请求。看到一些人意识到游戏与Java编程之间的区别真是太好了。但看到他们不知道如何激励他们的孩子，真的很让人伤心。我也不知道，但我可以试着猜一下。

    我不能代表每个人说话，但对我来说，为了“学习新东西”而学习根本不好玩。我不喜欢变聪明的过程。我喜欢构建东西。如果为了让新事物发生，我需要学习新东西，我会去做，我会喜欢学习，因为它会带来实际的结果：新的事物。

    例如，Kotlin已经在市场上有几年了。它是一种新的编程语言。我知道它可能最终取代Java，所以我需要学习它。但是我无法想象自己花几天时间阅读文档和学习它。这太无聊了。因为它没有任何意义。好吧，它会让我知道如何用Kotlin编码，但如果我已经能用Java编码了，那有什么意义呢？为了学习Kotlin，我需要一个新项目，然后用Kotlin编写。我将在使用它的同时学习这门语言。

    我的观点是，为了激励我自己学习Kotlin，你需要给我一个目标，在学习Kotlin的过程中，我将通过使用这些新知识来实现这个目标。也许，而且很可能，你的孩子也需要类似的东西：一个有形的激励因素。他们将在那里看到一些东西，当他们到达那里时会感到开心，这是在他们学习那些无聊的Java和Kotlin语言之后。对于一个九岁的孩子，这样的奖品可能是什么样子的呢？

    好吧，比如这些中的一个（也许不适用于所有年龄段，但你明白我的意思）：

    • 你能制作一个iPhone手机游戏并在苹果商店上发布吗？

    • 你能为我创建一个博客吗？

    • 你可以制作一个在线商店来销售我自制的饼干吗？

    • 你可以制作一个应用来帮助我追踪健身房的锻炼结果吗？

    • 你能在下一次黑客马拉松中赢得$1,000吗？

    • 你能获得 Oracle Java 证书吗？

    • 你能在学校的网站上找到安全漏洞吗？

    • 你能制造一个能清洁地板的机器人吗？

    实际上，这正是我了解计算机方面知识的方式。我大约在10岁左右开始编程。我总是有任务在面前，总是有一些有趣的事情可以为自己和周围的人做。换句话说，我一直注视着目标。我的许多项目已经不再存在了，尤其是我小时候和学生时期创建的那些项目（这让我感到很难过）。然而，结果是显而易见的：我现在在科技行业，能够使用Java进行编码。

    所以不要期望你的孩子喜欢学习。期望他们对建立东西和赚取利润感兴趣。作为他们的父母，你们的工作是帮助他们确定正确的目标—这是孩子们通常失败的地方，因为他们没有足够的经验来判断哪些目标更容易实现，哪些风险更大。我记得曾经试图创建一个能够像人类一样说话的机器人。那时我13岁。显然，在那个时候我需要帮助：有人告诉我这个目标并不是真正可以实现的，做一些更简单和更实际的事情会更好地运用我的努力。

    不要通过告诉他们需要学习什么、每天学习多少小时、上什么学校以及读哪些书籍来过分干预你的孩子。相反，像好的管理者对待员工一样对待他们：设定目标，并确保每个人都知道奖励和惩罚是什么。

    你的孩子需要你提供类似的东西。衡量他们的进展并帮助他们找到适当复杂和有趣的目标。此外，确保他们明白实现目标后会有什么好处，…嗯，你必须找到一种方法来惩罚他们的失败。不要因为他们愚蠢或懒惰而惩罚他们。唯一值得惩罚的是违背了实现结果的承诺。

    当然，这是他们对自己做出的承诺。
