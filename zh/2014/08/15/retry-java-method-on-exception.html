    How to Retry Java Method Call on Exception

    The following text is a partial translation of the original English article, performed by ChatGPT (gpt-3.5-turbo) and this Jekyll plugin:

    如果您有一个偶尔失败的方法，并且希望在抛出异常之前尝试几次，请使用@RetryOnFailure，它来自jcabi-aspects可以帮助您。例如，如果您要下载以下网页：

    @RetryOnFailure(
  attempts = 3,
  delay = 10,
  unit = TimeUnit.SECONDS
)
public String load(URL url) {
  return url.openConnection().getContent();
}

    这个方法调用只会在连续三次失败的执行之后，每次之间间隔十秒钟，才会抛出一个异常。

    This post explains how jcabi-aspects works with binary weaving. This mechanism integrates AspectJ with your code.

    当上面的示例中调用load()方法时，背后发生的是以下情况（伪代码）：

    while (attempts++ < 3) {
  try {
    return original_load(url);
  } catch (Throwable ex) {
    log("we failed, will try again in 10 seconds");
    sleep(10);
  }
}
