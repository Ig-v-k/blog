    这是我在Java代码中使用的一种注释规则：括号应该要么在一行的开头/结尾，要么在同一行成对出现。

    这种记号适用于任何编程语言（包括Java、Ruby、Python、C++、PHP等），其中方括号用于方法/函数调用。

    如果您遵循“配对括号”表示法，您的代码将如下所示：

    new Foo( // ends the line
  Math.max(10, 40), // open/close at the same line
  String.format(
    "hello, %s",
    new Name(
      Arrays.asList(
        "Jeff",
        "Lebowski"
      )
    )
  ) // starts the line
);

    显然，带有闭括号的行应该与带有其开放对的行具有相同的缩进级别。
