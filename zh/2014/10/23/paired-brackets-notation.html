    这是我在Java代码中使用的一条符号规则：括号应该要么在行首/行尾，要么在同一行中成对出现。

    这种符号规则适用于任何编程语言（包括Java、Ruby、Python、C++、PHP等），其中括号被用于方法/函数调用。

    如果你遵循这种“成对括号”符号规则，你的代码将会如下所示：

    new Foo( // ends the line
  Math.max(10, 40), // open/close at the same line
  String.format(
    "hello, %s",
    new Name(
      Arrays.asList(
        "Jeff",
        "Lebowski"
      )
    )
  ) // starts the line
);

    很明显，具有闭合括号的行应该从与具有其开放配对项的行相同的缩进级别开始。
