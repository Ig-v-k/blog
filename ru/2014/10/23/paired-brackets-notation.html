    Paired Brackets

    The following text is a partial translation of the original English article, performed by ChatGPT (gpt-3.5-turbo) and this Jekyll plugin:

    Вот правило записи, которое я использую в Java-коде: скобка должна либо начинать/заканчивать строку, либо быть сопряженной на той же строке.

    Обозначение применимо универсально к любому языку программирования (включая Java, Ruby, Python, C++, PHP и т. д.), где скобки используются для вызовов методов/функций.

    Вот как будет выглядеть ваш код, если вы будете использовать эту “Парную скобочную” нотацию:

    new Foo( // ends the line
  Math.max(10, 40), // open/close at the same line
  String.format(
    "hello, %s",
    new Name(
      Arrays.asList(
        "Jeff",
        "Lebowski"
      )
    )
  ) // starts the line
);

    Очевидно, что строка с закрывающей скобкой должна начинаться с того же уровня отступа, что и строка с открывающей парой.

    Ваша интегрированная среда разработки (IDE) будет отображать код следуя этой нотации (IntelliJ IDEA).
